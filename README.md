# LogisticRegressionFromScratch

This public repository implements Logistic Regression from Scratch on HADRON Collider Data, aiming to correctly classify
gamma rays vs. hadron noise... The class distribution is unequal leaning to 65% class 1 (actual gamma rays). In turn I will implement the entirety of logistic regression from scratch - refraining from the usage of 
"out-of-the-box" ML libraries that provide a LogisticRegression class (e.g. Scikit-learn).
